# Documentacao

Este diretorio concentra material funcional e operacional que nao faz parte do codigo executavel.

## Estrutura
- `bpo_financeiro/`: backlog, decisoes, processos e visao funcional do produto.

## Regras
- Evite criar documentacao solta na raiz do repositorio.
- Notas de ferramenta local, caches e arquivos de workspace nao devem ser versionados.
- Se um documento deixar de ser necessario ao fluxo de engenharia, prefira removelo ou movelo para um repositorio de conhecimento externo.
